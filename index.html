1
崴浤科技公司介紹與核心優勢

🏢 公司概況

崴浤科技成立於2012年，從最初約60人發展到現在約150人的規模，定位為完整的中大型網路行銷公司。

⚙️ 核心技術與研發

🎯 中央控管系統（獨特優勢）

市場上獨有的中央控管系統，能根據Google每次演算法更新，在10-30分鐘內同步更新所有客戶網站的架構、除錯及安全提升。

🔒 網站安全

  • Security Headers檢測幾乎都能達到A+等級
  • 通過台積電供應商網站資安規範評估
  • 通過「紅隊演練檢測攻擊」，阻擋率高達91%
  • 提供市場最高規格安全性
  • 額外增加Web Application Firewall (WAF) 系統防護

🌐 符合Google規範

  • 使用Google彰濱工業區數據中心主機
  • 採用Google建議的WFP圖片格式提高載入速度
  • 進行網頁壓縮

🌍 多節點主機

五大洲覆蓋

澳洲、美洲、非洲、亞洲等多節點主機服務

亞洲特別節點

日本、香港等地區專屬節點

優勢效果

國外訪客無需連回台灣主機，瀏覽速度更快

💾 備份機制

三層防護架構：Google主機 + 微軟主機（針對Bing和Yahoo） + 亞馬遜異地備份

3
崴浤科技網站設計方案

🎨 專案設計

費用：45萬元起
  • 專案團隊面對面溝通
  • 尋找國內外最新設計範例
  • 注重整體視覺、品牌意識
  • 動畫特效和頁面設計細膩度
  • 至少5-7頁設計

⚙️ 客製化設計

費用：10萬元內
  • 客製化但設計較不活潑
  • 2-3個區塊呈現重點
  • 可加購頁面、動畫特效
  • 可提升至接近專案設計質感

📋 公板設計

費用：最經濟
  • 選擇現有版型
  • 細節調整受限
  • 後台架構與其他方案相同
  • 適合傳統行業初期需求

🖼️ 設計細節

  • 圖片處理：提供10張版權圖用於Banner或分類圖
  • 產品圖：可協助後製，大量需額外收費
  • 頁面定義：一個網址通常算一頁，內容以A4計算

4
SEO服務方案

💰 服務費用

針對20組關鍵字優化：一年25萬元
續約價：20萬元/年

📊 服務內容

  • 優化20組關鍵字
  • 針對網站內頁進行優化
  • 目標進入搜尋結果前兩頁
  • 提供內容建議，協助網站內容更專業

📈 報告機制

整合報表

結合Google Analytics (GA) 和Google Search Console (GSC)的報表，透過Looker Studio呈現

數據內容

關鍵字表現、流量來源、國家及主要頁面排名

定期報告

每月提供20組關鍵字的排名變化報告

🌟 SEO優勢

  • 同時負責網站架構和行銷，能更全面優化
  • 程式碼、結構化數據底層調整
  • 圖片格式轉化（JPG轉WFP）
  • 提供部落格/問答文章撰寫服務（付費項目）
  • 每篇文章約需1-2週，參考國內外文獻及競業網站

🌍 多國語系與在地化

  • 網站支援多國語系切換
  • 設定語系標記，提升在地化搜尋效果
  • 網站可在中國大陸瀏覽
  • 目前無法協助提交至百度搜尋引擎（需實名制）

辰發公司業務補充

🏭 公司資訊

  • 地點：主要位於高雄，工廠設在前鎮
  • 主營業務：水處理（純水和廢水處理）
  • 客戶群：食品、電子產業、中小企業和化妝品等

💧 純水處理

根據客戶需求提供不同純度等級的水處理設備（RO水、純水、超純水），純度越高設備越貴、處理程序越複雜

🏭 廢水處理

根據水質和水量提供客製化的廢水處理方案，目標是達到排放標準

🔧 售後服務

濾心更換和保養維護是重要的售後服務環節

合作機會：崴浤科技客戶遍佈全省約2000家，未來可互相介紹客戶

🎯 後續行動項目

  • 崴浤科技將提供：
    • 不同方案的網站範例網址及價格範圍
    • 關鍵字行銷服務簡報(PPT)和報價（與網站建置分開）
    • AI文案生成功能的prompts範例和操作指引
  • 辰發公司將：內部討論各方案的適合性和預算規劃

📝 其他重要事項

  • 辰發公司代表對網站技術架構有一定了解，曾負責學校網站及系統建置
  • 辰發目前域名未導向臨時網站，可能影響曝光率
  • 臨時網站圖片多為網路抓取，存在版權問題，因此未正式上線
